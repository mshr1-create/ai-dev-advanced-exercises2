# ai-dev-advanced-exercises2

## プロジェクト概要
Gemini API を使用してニュース記事を取得し、AI でタグを生成し、Vue.js でカテゴリ別に表示する ニュースアプリです。

## セットアップ

### 必要な環境
- Python 3.8 以上
- `python-dotenv` パッケージ
- Gemini API キー

### インストール

```bash
# python-dotenv のインストール
py -m pip install python-dotenv

# .env ファイルの作成（プロジェクト直下）
# GEMINI_API_KEY=your_api_key_here
```

## 使用方法

### ニュースデータの更新方法

#### Python スクリプト実行
```bash
py rss.py
```
実行後、`all_topics.json` に最新のニュース情報が保存されます（実行時間: 約1～2分）。

### Webサーバーの起動

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセスします。

## 機能

### RSS 取得 (`rss.py`)
- 複数のニュースフィード（スポーツ、IT、ビジネス）から記事を取得
- 記事のメタデータ（タイトル、リンク、説明、本文、公開日時）を抽出
- ネットワーク障害に対応したリトライ機能付き

### Gemini API タグ生成
- 各記事の内容に基づいて、Gemini 2.5 Flash を用いてタグを生成
- 無料枠対応：レート制限（15秒/回）を実装
- API エラー時も処理継続、ログ出力

### Vue.js フロントエンド (`index.html`)
- カテゴリ（政治、経済、IT、スポーツ、社会、企業）別にニュースを絞り込み
- 選択状態を視覚的に強調表示
- リロード時に初期カテゴリ（全表示）を正しく復帰
- 記事タイトルから元のニュースサイトへのリンク機能

## リアルタイム更新の仕組み

**ニュース内容は手動更新です。以下の手順で最新情報を取得できます:**

1. ターミナルで `py rss.py` を実行
2. スクリプトが実行中の間、待機（約1～2分）
3. スクリプト完了後、ブラウザで `F5` キーを押してページをリロード
4. 最新のニュース情報が表示されます

## トークン使用量の最適化

- フィード数: 3
- 1フィードあたりの取得件数: 4件（最大12件）
- API 呼び出し間隔: 15秒/回
- 1回実行あたりのトークン消費量: 実行環境により変動（無料枠を考慮）

## トラブルシューティング

### ニュースが表示されない場合
- ブラウザのコンソール（F12）でエラーを確認してください
- `all_topics.json` が存在するか確認してください
- ローカルサーバーで実行しているか確認してください（`file://` プロトコルでは CORS エラーが発生します）

### API エラーが出る場合
- `.env` に `GEMINI_API_KEY` が正しく設定されているか確認
- API クォータを確認（無料枠は 1分あたり 5リクエストに制限）
- 必要に応じてスクリプト実行を間隔を開けて実行してください

## 技術スタック

- **バックエンド**: Python 3, Gemini 2.5 Flash API
- **フロントエンド**: Vue.js 2, CSS3
- **データ形式**: JSON
